# beamer-visual-style

Usa esta skill para aplicar o ajustar el estilo visual de presentaciones Beamer en este repositorio.

## Reglas visuales

- Usar el tema `jamcy`.
- Mantener consistencia visual entre slides.
- Usar colores definidos por el tema.
- Evitar estilos improvisados dentro de cada presentación.
- Evitar redefinir colores, tipografías o bullets localmente sin necesidad.
- Poner cambios globales en `themes/jamcy/` cuando deban afectar a varios decks.

## Alcance

Los ajustes visuales deben ser simples, editables y compatibles con Beamer, `lualatex`, MiKTeX y Windows.
