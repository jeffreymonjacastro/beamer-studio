# beamer-refactor-slides

Usa esta skill para mejorar slides existentes sin cambiar el contenido técnico de forma silenciosa.

## Principios

- Reducir texto excesivo.
- Convertir párrafos largos en bullets claros.
- Separar una slide cargada en varias slides.
- Mantener una idea principal por frame.
- Usar `block`, `alertblock` o `exampleblock` cuando ayuden a jerarquizar.
- Conservar notación y afirmaciones técnicas salvo que el usuario pida cambios.

## Reglas

- No alterar el contenido técnico sin avisar.
- No reemplazar resultados, métricas o fórmulas por aproximaciones no pedidas.
- Mantener el estilo del tema `jamcy`.
- Preferir cambios locales y legibles dentro de `sections/`.
